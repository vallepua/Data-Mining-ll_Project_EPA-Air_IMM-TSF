# Data-Mining-ll_Project_EPA-Air_IMM-TSF:

This project investigates the challenges of forecasting using "messy" real-world data that includes both numerical sensors data and unstructured textual data. Using the IMM-TSF (Irregular Multimodal Multivariate Time-Series Forecasting) framework.

We focus on the EPA-Air dataset. In Track A, we successfully replicated the baseline performance metrics from the original research, confirming that integrating textual data reduces prediction error. In Track C, we propose an architectural extension to the modality fusion module to further enhance the model's ability to handle asynchronous data streams.
